# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using machine learning models on the Kaggle **House Prices - Advanced Regression Techniques** dataset.

The project demonstrates the complete machine learning workflow from data preprocessing to model training and prediction.
